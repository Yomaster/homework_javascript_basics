Дан код: 
            
                var a = 1, b = 1, c, d;
                c = ++a; alert(c);           // 2
                d = b++; alert(d);           // 1
                c = (2+ ++a); alert(c);      // 5
                d = (2+ b++); alert(d);      // 4
                alert(a);                    // 3
                alert(b);                    // 3
Почему код даёт именно такие результаты?
Ответ:
Потому что преинкремент ++a вначале увеличивает значение пременной a на 1, а затем это новое значение передается переменной с. Значение же переменной b с использованием постинкремента b++ сначала передается пременной d, а уже потом происходит увеличение значения переменной b на 1.
